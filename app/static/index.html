Abbreviation Resolution

Tự động phân giải từ viết tắt bằng LLM (llama.cpp) và xuất dữ liệu tương thích trình gán nhãn.
Mở trình gán nhãn →

1Tải dữ liệu

Văn bản thô (JSON list of dict, văn bản ở trường input) và từ điển viết tắt (JSON list, các trường word/type/meaning/label).
Chọn khoảng mẫu để xử lý theo chỉ số (0-based, bao gồm cả hai đầu). Ví dụ bắt đầu=0, kết thúc=49 → xử lý 50 mẫu đầu. Sau khi bị timeout, đặt index bắt đầu = next_index của checkpoint để chạy tiếp.

2Cấu hình LLM (llama.cpp)

Trỏ tới llama-server (OpenAI-compatible). Các giá trị mặc định nạp từ config.json; bạn có thể sửa và bấm "Lưu làm mặc định". Hoặc bật "Chạy thử" để xem pipeline mà không cần model.

3Chạy phân giải

Mỗi văn bản chỉ gọi LLM một lần. Từ label 0 gán trực tiếp; từ label 1 để LLM chọn theo ngữ cảnh (hoặc "không nghĩa phù hợp").

Checkpoints đã lưu

Tự động lưu sau mỗi N mẫu (cấu hình ở bước 2). Dùng để khôi phục khi bị timeout: tải JSON kết quả tạm, hoặc đặt index bắt đầu = next_index để chạy tiếp.
